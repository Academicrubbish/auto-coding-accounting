## 测试覆盖范围

自动化测试应覆盖以下场景：

### 关键路径测试
1. **新用户首次使用** → 游客浏览 → 触发登录 → 完成记账
2. **完整记账流程** → 查看统计 → 编辑交易 → 删除交易
3. **账户管理** → 创建账户 → 设为默认 → 删除账户
4. **分类管理** → 创建分类 → 使用分类 → 删除分类

### 边界条件测试
1. **余额计算** - 支出/收入/编辑/删除后的余额正确性
2. **数据隔离** - 用户数据互不干扰
3. **权限验证** - 未登录无法执行写操作
4. **分页加载** - 大量数据的加载性能

### 异常场景测试
1. **网络异常** - 云函数调用失败处理
2. **数据缺失** - 默认数据降级显示
3. **权限拒绝** - token 过期处理
